Difference between Map, filter, forEach and find?

Map is a function in programming that takes a sequence of data, applies a given operation or transformation to each element of the sequence, and returns a new sequence with the transformed values.
Filter is used to create a new array based on a specific condition.
ForEach executes a provided function once for each element in an array.
Find returns the value of the first element in an array

Difference between localStorage, Session, Cookie?

localStorage stores data permanently
sessionStorage stores data temporarily until the browser window or tab is closed.
Cookies are small pieces of data stored by a website in a user's browser.

What is event loop? how does it works?

Event loop in JavaScript is a mechanism that allows the runtime to efficiently handle asynchronous operations.
Works: It works by continuously checking the call stack and the message queue.
Examples: 

console.log('Start');
setTimeout(() => {
  console.log('Timeout');
}, 0);
Promise.resolve().then(() => {
  console.log('Promise');
});
console.log('End');
